# EduEnroll Platform
A student enrollment system built with PHP and MySQL.

## Features
- Admin dashboard for managing students and courses
- CRUD operations (Create, Read, Update, Delete)
- Responsive design

## How to Run Locally
1. Install XAMPP
2. Copy project folder into htdocs
3. Import `database/enrollment_db.sql` into phpMyAdmin
4. Run `http://localhost/eduenroll`

 ## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP
